# bolg-article-api-java-study
使用java开发了的个人小笔记站的API接口（计划后期用来做VUE的服务端接口）。接口目前有文章列表接口；文章详情扩接口；评论接口；以及文章分类接口。通过此项目能熟悉JAVA进行WEB开发的流程。
页面在线示例地址：http://api.04007.cn/  适用于做博客网站服务端API接口程序，欢迎使用。
